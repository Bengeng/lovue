# lovue

### 如何跑起来：

```
git clone git@github.com:Lovuey/lovue.git
cd lovue
npm install
npm run serve
```
### 提交说明

master 分支是当前最新的稳定版本，提交代码不能提交到 master 分支；
dev 分支为开发分支，可以在这个分支提交代码；

### 代码规范

- 文件名首字母小写；

### 目录说明

- asset 资源文件，比如图片；
- const 常量
- pages 页面
- utils 工具

### 线上
```
npm run build
```
